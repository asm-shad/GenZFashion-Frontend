## Project Overview

**GenZ Fashion** is a modern and scalable **e-commerce frontend application** built with **React 19**, **Vite**, and **Redux Toolkit**.  
The application delivers a fast, responsive, and interactive shopping experience using **Tailwind CSS** for styling and a component-driven architecture.

The project is designed with performance, maintainability, and real-world e-commerce workflows in mind, making it suitable for production use.

---

## Key Features Implemented

- **Product & Order Management**  
  Browse products, view product details, add items to cart, and place orders.

- **Cart & Checkout Flow**  
  Persistent cart experience with seamless checkout integration.

- **Payment Integration**

  - **Stripe** for card-based payments
  - **PayPal** for alternative payment support

- **Role-Based Access Control (RBAC)**  
  Supports roles such as **Admin**, **Product Manager**, and **Customer Support** with controlled access.

- **State Management**  
  Centralized and predictable state management using **Redux Toolkit**.

- **Routing & Navigation**  
  Client-side routing handled with **React Router DOM**.

- **Notifications & Feedback**  
  Real-time user notifications using **Sonner**.

- **Responsive UI**  
  Mobile-first, fully responsive design using **Tailwind CSS**.
